# Activity Tracker Frontend

Next.js application for the Activity Tracker.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Axios
- Next-Themes (Dark Mode)

## Features
- **Activity Management**: Record and view activities with filtering and pagination.
- **User Management**: Create and view users.
- **Responsive Design**: Optimized for mobile and desktop views.
- **Dark Mode**: Toggle between light and dark themes.
- **Modern UI**: Built with Tailwind CSS and Geist fonts.

## Running

```bash
npm install
npm run dev
```

## Structure
- `app/`: Pages and layouts
- `components/`: Reusable UI components
- `services/`: API integration
- `types/`: TypeScript definitions
